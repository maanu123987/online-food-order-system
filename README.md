# Online Food Order System

## Overview
A modern multi-page online food ordering application built with React, React Router, Express, MySQL, and JWT authentication.

## Features
- Multi-page React frontend with React Router
- User auth, protected routes, and admin login
- Menu, food details, cart, checkout, payment, order success, and history
- Admin dashboard for foods, categories, orders, users, and reports
- MySQL schema and backend API routes

## Run Instructions
1. Create a MySQL database and import backend/schema.sql
2. Update backend/.env with your DB credentials
3. Install frontend deps: cd frontend && npm install
4. Install backend deps: cd backend && npm install
5. Start backend: npm run dev
6. Start frontend: npm start
